python安装 django ， setuptools ， pip 、easy_install 等插件

  1. python是1.4.1版本 需要将 python_home, 和 python_home/scripts, 这两个目录添加到path中
  2. easy_install 应该会自带
  3. 下载 setuptools 并 ez_setup.py install    
  4. 安装 pip  , get-pip.py install 
  5. 安装 ipython  setup.py install 
  6. 安装django  pip install Django==1.5.2
  7. ok


就可以使用python中各种插件了
