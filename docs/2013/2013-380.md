 <li> 安装 ipython &nbsp;<a href="http://archive.ipython.org/release/0.13.2/ipython-0.13.2.tar.gz">setup.py</a> install&nbsp; </li> 
 <li> 安装django &nbsp;pip install Django==1.5.2 </li> 
 <li> ok </li>
