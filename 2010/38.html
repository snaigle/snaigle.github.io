上周末用 grub 安装了ubuntu 10.10

上周末 终于安装了 ubuntu10.10  ，安装的是 kanas-ubuntu-10.10-desktop-i386.iso 中文定制版,  算是减小了入门的难度吧，里面默认安装了中文字体输入法，flash，播放器，chm等，感兴趣的可以安装试试。

本来打算用wubi安装的，结果这个iso没提供这个选项，又没条件去刻一张dvd，只好查资料准备采用 grub安装，

安装教程 网上有很多，我用的是这个。

xp下安装和配置启动grub，以及grub【windows版】下载 看：http://www.ylmf.net/read.php?tid=1329214

 

下载并解压缩后，将目录中的 grldr , grldr.mbr ， grub.exe 三个文件复制到 C 盘根目录下

2、Grub安装Ubuntu在下载好的 ubuntu 10.10 系统 iso 文件中 ， casper 文件夹目录下，找到 vmlinuz 、 initrd.lz （注意： ubuntu9.04 文件名为 initrd.gz ）解压，并复制到 C 盘根目录下（无需解压整个 casper 文件夹，整个 casper 文件夹很大）

将iso 文件也放到C盘根目录上。

 

新建 C:\menu.lst文件 打开C:menu.lst，在里面增加下面一段话。 

    title Boot ubuntuISO    find --set-root /ubuntu-9.04-desktop-i386.iso    map --mem /ubuntu-9.04-desktop-i386.iso (hd32)    map --hook    chainloader (hd32)    boot

保存并退出。


 


六、下载ubuntu-9.04-desktop-i386.iso镜像文件，保存至X:\目录下，X代表xp下的任何一个盘符。记得一定要放到根目录下。至于为什么，大家应该都清楚吧。 

[编辑 ] 如果你下载的是DVD镜像或打算下载DVD镜像

前面四步完全和CD镜像相同，下面将从第五步开始讲述
五、配置C:\menu.lst文件 打开C:menu.lst，在里面增加下面一段话。 

 title Boot UbuntuDVDfind --set-root /ubuntu-10.10-dvd-i386.isokernel vmlinuz iso-scan/filename=/ubuntu-10.10-dvd-i386.iso boot=casper roinitrd initrd.gzboot

六、把ubuntu-10.10-dvd-i386.iso放到一个分区的根目录下，再用虚拟光驱挂上，把光盘目录里的 /casper/initrd.gz和 /casper/vmlinuz 两个文件拷到和ubuntu-10.10-dvd-i386.iso同一分区下
下载就可以重启进入Grub选择Boot UbuntuDVD来安装Ubuntu了

  

进入 livecd 后 在终端 执行   sudo umount -l /isoservice 将虚拟光驱卸载，要不然 你利用分区安装 时 会报错

 

安装完成后，会发现启动菜单上没有xp， 这个不用急，只需在终端 运行 sudo update-grub命令即可，然后重启就能在开机时看到 xp的选项了
